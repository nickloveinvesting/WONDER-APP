## Testing Methodology

### What to Measure

**Primary Metrics:**
- Click-through rate (CTR)
- Sign-up conversion rate
- Time on page
- Scroll depth

**Secondary Metrics:**
- Bounce rate
- Return visits
- Social shares
- Message retention (recall surveys)

**Qualitative Metrics:**
- User feedback on clarity
- Questions asked (indicates confusion)
- Emotional response (surveys)
- Brand perception

### How to Test

**Quantitative:**
- A/B tests on landing pages (Google Optimize, etc.)
- Email subject line tests
- Social media ad variations
- Different platform-specific approaches

**Qualitative:**
- User interviews: "What does this message mean to you?"
- Clarity surveys: "Is this clear?" "Is this compelling?"
- Preference tests: "Which resonates more?"
- Five-second tests: "What do you remember?"

**Mixed Methods:**
- Analytics + exit surveys
- A/B test + user interviews with each variant
- Quantitative results + qualitative explanation

### Sample Sizes

**Minimum for statistical significance:**
- ~1,000 visitors per variant for conversion tests
- ~100 responses for survey-based qualitative
- ~10-15 interviews for deep qualitative insights

**Time frames:**
- 1-2 weeks minimum for traffic-based tests
- Consider day-of-week and time-of-day effects
- Run longer for low-traffic scenarios

## Notes on Messaging Testing

**Remember:**
- Test one variable at a time (except for combo tests)
- Run tests long enough for statistical significance
- Consider context (platform, audience, timing)
- Qualitative insight explains quantitative results
- Winners in one context may not win in another
- Keep testing—messaging optimization never stops

**Common Mistakes:**
- Testing too many variants simultaneously (dilutes traffic)
- Stopping tests too early (false positives)
- Ignoring qualitative feedback (numbers without understanding)
- Not documenting learnings (repeating tests)
- Testing everything equally (prioritize high-impact)

**Success Indicators:**
- Clear winner emerges (not just noise)
- Results align with qualitative feedback
- Learnings inform future messaging
- Conversion rates improve systematically
- Brand voice becomes clearer over time
