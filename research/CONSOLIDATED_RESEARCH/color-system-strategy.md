# Color System Strategy

**Research Date:** November 2025
**Agent:** Agent 6 - Visual Design Language & Component System
**Focus:** Color palette, psychology, accessibility, and functional color usage

---

## Executive Summary

The current ARGUED color palette (Navy #1A3A52, Brown #8B6F47, Cream #F5F3F0, Black #1C1C1C) is sophisticated and intellectually credible, but **leans competitive rather than collaborative**. This document evaluates the existing palette against conversation-first goals and recommends an evolved color system.

**Key Findings:**
- **Navy (#1A3A52)** is trustworthy but feels corporate/formal
- **Brown (#8B6F47)** is warm but associated with competition/winning
- **Cream (#F5F3F0)** is excellentâ€”warm, sophisticated, accessible
- **Overall palette** works for debate platform, needs evolution for conversation platform

**Recommendations:**
1. **Keep** cream background (excellent choice)
2. **Soften** navy to warmer blue-gray
3. **Recontextualize** brown from "winning" to "depth/wisdom"
4. **Add** collaborative accent (sage green or warm purple)
5. **Expand** functional colors for conversation depth, topic categories

**Accessibility Status:**
Current palette meets WCAG AA standards but needs dark mode refinement and expanded semantic colors.

## Accessibility Compliance

### WCAG 2.1 Standards

**Level AA Requirements (Minimum):**
- Normal text: 4.5:1 contrast ratio
- Large text (18px+): 3:1 contrast ratio
- UI components: 3:1 contrast ratio

**Level AAA Requirements (Goal):**
- Normal text: 7:1 contrast ratio
- Large text: 4.5:1 contrast ratio

### Color Blind Accessibility

**Types of Color Blindness:**
- Deuteranopia (red-green, 5% of males)
- Protanopia (red-green, 1% of males)
- Tritanopia (blue-yellow, rare)

**Current Palette Issues:**
- Brown and green may be indistinguishable (red-green color blindness)
- Need to not rely solely on color for meaning

**Solutions:**
1. **Always pair color with icon/text label**
   - Don't use color alone to convey meaning
   - Example: "Growth achieved âœ“" (text + icon + color)

2. **Use patterns in addition to color**
   - Striped, dotted, solid patterns
   - Different shapes (circle, square, triangle)

3. **Test with simulators**
   - Chrome DevTools (emulate vision deficiencies)
   - Color Oracle (desktop app)
   - Sim Daltonism (Mac)

4. **Provide colorblind-friendly mode**
   - High-contrast colors
   - Stronger visual indicators
   - User preference setting
