   - Question closure feels like rejection â†’ Don't return

   - **Ponder advantage**: No downvotes, no closure, collaborative not competitive

2. **Format Rigidity**
   - Only Q&A format â†’ Limits discussion types
   - Must have "correct answer" â†’ Excludes exploratory philosophy
