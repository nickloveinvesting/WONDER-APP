## TESTING METHODOLOGY

### Test 1: First Impression Test (5 seconds)

**Method:** Show name for 5 seconds, ask what they think it's about

**Evaluation Criteria:**
- Did they guess correctly?
- What emotions did it evoke?
- Did it sound tech/academic/social/gaming?

### Test 2: Recall Test (24 hours later)

**Method:** Mention name in context, ask them to recall it next day

**Evaluation Criteria:**
- Can they remember it?
- Can they spell it?
- Did they confuse it with something else?

### Test 3: Verbal Sharing Test

**Method:** Ask them to tell friend about app by name only

**Evaluation Criteria:**
- Would they feel confident saying it?
- Does it require explanation?
- Is pronunciation intuitive?

### Test 4: Search Behavior Test

**Method:** Ask them to search for app without showing spelling

**Evaluation Criteria:**
- Can they find it?
- What do they type?
- Alternative spellings tried?

### Test 5: Emotional Association Test

**Method:** Rate feelings on scale (intimidated, excited, curious, confused, indifferent)

**Evaluation Criteria:**
- Positive vs negative emotions
- Excitement level
- Accessibility perception
