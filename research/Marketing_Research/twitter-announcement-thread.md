## TWITTER THREAD BEST PRACTICES

### DO:
- Start with a strong hook (problem or curiosity)
- Use line breaks for readability
- Include visuals (screenshots, GIFs)
- End with clear CTA
- Number your tweets (1/12 format)
- Ask questions to drive engagement
- Reply to every comment quickly

### DON'T:
- Make the first tweet too long (loses hook)
- Use jargon or academic language
- Forget to include the link
- Post all at once (thread throughout)
- Ignore engagement
- Over-promote (balance value and promotion)

### Posting Strategy:
- **Best times:** 9-11 AM EST or 5-7 PM EST (workday bookends)
- **Best days:** Tuesday-Thursday
- **Avoid:** Late night, weekends
- **Hashtags:** 2-3 max (#Philosophy #AI #Debate)
- **Engage:** Reply to comments within 5 minutes

### Visual Strategy:
- Tweet 1: No visual (strong text hook)
- Tweet 2-3: Problem illustration
- Tweet 4-5: Solution screenshot
- Tweet 6-7: Product demo GIF
- Tweet 8-9: Results/features
- Tweet 10: CTA with screenshot
